1. Context

The McCullagh (1980) proportional-odds model parametrises an ordered categorical response \(y \in \{1, \ldots, K\}\) via cumulative logits

\[ \operatorname{logit}\bigl(\Pr(y \le k \mid x)\bigr) \;=\; \theta_k - x^{\top}\beta,\qquad k = 1, \ldots, K - 1, \]

with thresholds \(\theta_1 < \ldots < \theta_{K-1}\) shared across categories and a single slope vector \(\beta\). The Tutz (1990) §3.2 joint Newton iteration updates \((\beta, \theta)\) simultaneously using the closed-form block-diagonal Hessian \(H = \operatorname{diag}(H_{\beta\beta}, H_{\theta\theta}) + H_{\beta\theta}\) (McCullagh 1980 §2.5).

ds.vertOrdinalJointNewton runs the K = 2 OT-Beaver pipeline: each iteration’s \(H_{\beta\theta}\) cross-block is computed via Beaver vector multiplication; \(\theta\)-block recovery and the post-Newton refinement step keep the federated estimator anchored to the centralised MASS::polr solution.

Non-disclosure invariants

Invariant Statement
D-INV-1 Per-patient covariates never leave their origin server.
D-INV-2 Per-row \(x_i^{\top}\beta\) never reconstructed; cumulative scores stay as additive shares.
D-INV-3 Per-patient cumulative indicators \(\mathbf{1}\{y_i \le k\}\) live only on the label server.
D-INV-4 Threshold \(\theta\) updates exchange aggregate \(H_{\beta\theta}\) Beaver products only.
D-INV-5 Cross-server messages signed Ed25519 against dsvert.trusted_peers.

Dataset and partition

The MASS birthwt dataset (Hosmer & Lemeshow, 2000) — 189 women with birth weight outcome plus risk covariates. We discretise bwt into three ordinal classes (low / medium / high birth-weight tertiles) for a balanced ordinal target. Cumulative-probability indicator columns low_leq, med_leq, high_leq are pre-computed on the label server (the canonical cumulative_template input expected by ds.vertOrdinalJointNewton).

Server Columns Role
s1 patient_id, age, lwt, smoke Maternal demographics + risk
s2 patient_id, bwt_cls, low_leq, med_leq, high_leq Ordinal outcome + cumulative indicators (label)

2. Data preparation

library(dsVertClient)
library(DSLite)
library(DSI)
library(MASS)
library(ggplot2)
data("birthwt", package = "MASS")
bw <- birthwt
bw$patient_id <- sprintf("BW%03d", seq_len(nrow(bw)))
brks <- quantile(bw$bwt, c(0, 1/3, 2/3, 1), na.rm = TRUE)
bw$bwt_cls <- factor(
  cut(bw$bwt, breaks = brks, include.lowest = TRUE,
      labels = c("low", "med", "high")),
  levels = c("low", "med", "high"), ordered = TRUE)
bw$low_leq  <- as.integer(as.integer(bw$bwt_cls) <= 1L)
bw$med_leq  <- as.integer(as.integer(bw$bwt_cls) <= 2L)
bw$high_leq <- 1L
pooled <- bw[, c("patient_id",
                 "age", "lwt", "smoke",
                 "bwt_cls",
                 "low_leq", "med_leq", "high_leq")]

3. Federated K=2 ordinal joint Newton

fit_ds <- ds.vertOrdinalJointNewton(
  bwt_cls ~ age + lwt + smoke,
  data = "DA",
  levels_ordered = c("low", "med", "high"),
  cumulative_template = "%s_leq",
  max_outer = 8L, tol = 1e-6,
  verbose = FALSE, datasources = conns)

5. Comparison and verdict

ds_b <- as.numeric(fit_ds$beta_po_joint %||% fit_ds$coefficients)
rf_b <- as.numeric(coef(fit_ref))
nm   <- names(coef(fit_ref))
delta <- data.frame(
  coef   = nm,
  dsVert = ds_b[seq_along(rf_b)],
  polr   = rf_b)
delta$abs <- abs(delta$dsVert - delta$polr)
delta
#>    coef   dsVert     polr     abs
#> 1   age -0.02534  0.02534 0.05069
#> 2   lwt -0.01423  0.01412 0.02835
#> 3 smoke  0.68827 -0.67865 1.36692

max_abs    <- max(delta$abs)
sigma_min  <- min(sqrt(diag(vcov(fit_ref))))
sigma_ratio <- sigma_min / max_abs

c(max_abs_delta = max_abs,
  sigma_min     = sigma_min,
  sigma_ratio   = sigma_ratio)
#> max_abs_delta     sigma_min   sigma_ratio 
#>      1.366921      0.004825      0.003530
ggplot(delta, aes(polr, dsVert)) +
  geom_abline(slope = 1, intercept = 0, linetype = "dashed",
              colour = "grey40") +
  geom_point(size = 2.5) +
  labs(x = expression(hat(beta)[polr]),
       y = expression(hat(beta)[dsVert])) +
  theme_minimal(base_size = 11)
