1. Context

ds.vertOrdinalJointNewton fits the proportional-odds model against the MASS::polr target. The current strict path keeps class masks, cumulative probabilities, reciprocal products, beta/theta scores and finite-difference Hessian probes inside Ring127 additive shares until only low-dimensional aggregate scores are opened.

The full DSLite validation is intentionally not re-run during vignette rendering because each K mode takes about 11 minutes on the deterministic n = 24 fixture. This vignette loads the checked DSLite evidence generated by the method harness and records the exact commands needed to reproduce the caches.

Non-disclosure invariants

Invariant Statement
D-INV-1 Per-patient covariates never leave their origin server.
D-INV-2 Per-patient cumulative probabilities, weights and residuals are never reconstructed in plaintext.
D-INV-3 Class masks are encrypted Ring127 additive shares between DCF parties.
D-INV-4 K >= 3 non-DCF servers export encrypted additive shares only.
D-INV-5 Returned-object audit rejects numeric vectors of patient length.

# From the dsvert-paper repository root.
Rscript scripts/validate_method_ordinal_joint.R \

4. Accuracy

summary_table <- result$summary
summary_table[, c("K", "n", "max_abs_cumprob_delta",
                  "warm_max_abs_cumprob_delta", "improvement",
                  "grad_beta_inf", "grad_theta_inf", "runtime_s",
                  "returned_object_audit")]
#>    K  n max_abs_cumprob_delta warm_max_abs_cumprob_delta improvement
#> K2 2 24          0.0004281331                 0.04985902  0.04943089
#> K3 3 24          0.0005697185                 0.04986504  0.04929532
#>    grad_beta_inf grad_theta_inf runtime_s returned_object_audit
#> K2    0.02039956   0.0002444913   655.075                  TRUE
#> K3    0.01982879   0.0002276716   657.098                  TRUE
plot_df <- data.frame(
  K = factor(summary_table$K),
  strict = summary_table$max_abs_cumprob_delta,
  warm = summary_table$warm_max_abs_cumprob_delta
)
plot_df <- reshape(
  plot_df, varying = c("strict", "warm"),
  v.names = "max_abs_cumprob_delta",
  timevar = "route", times = c("strict_joint", "warm"),
  direction = "long")
ggplot(plot_df, aes(K, max_abs_cumprob_delta, fill = route)) +
  geom_col(position = "dodge") +
  scale_y_log10() +
  labs(x = "K mode", y = "max absolute cumulative-probability delta") +
  theme_minimal(base_size = 11)

Strict proportional-odds cumulative-probability error versus warm baseline.

The strict joint route reaches maximum cumulative-probability gaps of 5.697e-04 across K = 2 and K >= 3, while the warm route is around 4.987e-02 on the same fixture. Both returned-object audits pass and both verdicts are STRICT.
