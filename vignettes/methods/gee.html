1. Context

Generalised estimating equations (GEE) extend GLM to clustered or longitudinal outcomes by specifying a working correlation structure \(R(\alpha)\) on within-cluster errors and solving the quasi-score

\[ U(\beta) \;=\; \sum_{i=1}^K D_i^{\top} V_i^{-1}\bigl(y_i - \mu_i(\beta)\bigr) \;=\; 0 \]

where \(D_i = \partial \mu_i / \partial \beta\), \(V_i = A_i^{1/2} R(\alpha) A_i^{1/2}\), and \(A_i\) is the diagonal of GLM-implied marginal variances. Liang & Zeger (1986) showed that \(\hat\beta\) is consistent and asymptotically normal even under mis-specified \(R(\alpha)\), with covariance estimated by the sandwich \(\widehat{\operatorname{Cov}}(\hat\beta) = B^{-1} M B^{-1}\) where \(B = \sum D_i^{\top} V_i^{-1} D_i\) and \(M = \sum D_i^{\top} V_i^{-1} (y_i - \mu_i)(y_i - \mu_i)^{\top} V_i^{-1} D_i\).

In the federated K = 3 setting the design partitions across servers; the cluster id and outcome live on the label server. ds.vertGEE runs an alternating fixed-effects IRLS / working-correlation moment update and returns the sandwich-corrected coefficients consistent with geepack::geeglm.

Non-disclosure invariants

Invariant Statement
D-INV-1 Per-patient covariates never leave their origin server.
D-INV-2 Per-row \(x_i^{\top}\beta\) is never reconstructed in plaintext.
D-INV-3 Per-patient residuals are never revealed; only aggregate \(D_i^{\top} V_i^{-1} (y_i - \mu_i)\) blocks are.
D-INV-4 Cluster identifiers are visible only on the label server.
D-INV-5 All cross-server messages signed Ed25519, peer pinned via dsvert.trusted_peers.

Dataset and partition

The Pothoff–Roy Orthodont dataset (Pinheiro & Bates, 2000 §1.4) records four longitudinal dental measurements per child across ages 8, 10, 12, 14 — 27 children × 4 visits = 108 rows. The repeated measures within child are clustered, motivating GEE.

Server Columns Role
s1 patient_id, age Time covariate
s2 patient_id, sex_num Sex covariate
s3 patient_id, cluster_id, distance Cluster id + outcome

5. Comparison and verdict

nm <- intersect(names(fit_ds$coefficients), names(coef(fit_ref)))
delta <- data.frame(
  coef    = nm,
  dsVert  = as.numeric(fit_ds$coefficients[nm]),
  geeglm  = as.numeric(coef(fit_ref)[nm]))
delta$abs <- abs(delta$dsVert - delta$geeglm)
delta
#>          coef  dsVert  geeglm       abs
#> 1 (Intercept) 15.3860 15.3857 0.0003249
#> 2         age  0.6601  0.6602 0.0000396
#> 3     sex_num  2.3212  2.3210 0.0001868

max_abs    <- max(delta$abs)
sigma_min  <- min(sqrt(diag(summary(fit_ref)$cov.scaled)))
sigma_ratio <- sigma_min / max_abs

c(max_abs_delta = max_abs,
  sigma_min     = sigma_min,
  sigma_ratio   = sigma_ratio)
#> max_abs_delta     sigma_min   sigma_ratio 
#>     0.0003249           NaN           NaN
ggplot(delta, aes(geeglm, dsVert)) +
  geom_abline(slope = 1, intercept = 0, linetype = "dashed",
              colour = "grey40") +
  geom_point(size = 2.5) +
  labs(x = expression(hat(beta)[geeglm]),
       y = expression(hat(beta)[dsVert])) +
  theme_minimal(base_size = 11)

Federated GEE coefficients vs centralised geepack::geeglm fit. Dashed identity line.

The federated GEE estimator agrees with geepack::geeglm to a maximum absolute slope deviation of 3.25e-04, giving a σ-ratio of NaN× against the smallest sandwich-corrected standard error of the centralised fit. The clustered-error working correlation \(R(\alpha)\) is recovered within the Ring63 Beaver fixed-point envelope.
