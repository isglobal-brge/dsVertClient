1. Context

ds.vertMultinomJointNewton fits a true joint softmax model rather than the older one-vs-rest warm approximation. The strict route uses Ring127 share-domain softmax probabilities and residuals, with a Bohning majorization step based on low-dimensional aggregate Gram information. K >= 3 uses outcome and fusion DCF parties; non-DCF feature servers send encrypted additive shares only.

The full DSLite validation is intentionally not re-run during vignette rendering because each K mode takes about 14 minutes on the deterministic n = 50 fixture. This vignette loads the checked DSLite evidence generated by the method harness and records the exact commands needed to reproduce the caches.

Non-disclosure invariants

Invariant Statement
D-INV-1 Per-patient covariates never leave their origin server.
D-INV-2 Per-patient softmax probabilities and residuals remain Ring127 additive shares.
D-INV-3 The client receives aggregate gradients, the aggregate Bohning Gram and fitted coefficients only.
D-INV-4 K >= 3 non-DCF servers do not receive labels, probabilities, residuals or fitted values.
D-INV-5 Returned-object audit rejects numeric vectors of patient length.

2. Reproduction Commands

# From the dsvert-paper repository root.
Rscript scripts/validate_method_multinom_joint.R \
  --K 2 --n 50 --max-outer 30 --tol 1e-4 --privacy-level 5

Rscript scripts/validate_method_multinom_joint.R \
  --K 3 --n 50 --max-outer 30 --tol 1e-4 --privacy-level 5

4. Accuracy

summary_table <- result$summary
summary_table[, c("K", "n", "max_abs_prob_delta",
                  "warm_max_abs_prob_delta", "joint_improvement",
                  "max_abs_coef_delta", "runtime_s",
                  "returned_object_audit")]
#>    K  n max_abs_prob_delta warm_max_abs_prob_delta joint_improvement
#> K2 2 50       0.0007368566               0.2443136         0.2435767
#> K3 3 50       0.0007657740               0.2443057         0.2435399
#>    max_abs_coef_delta runtime_s returned_object_audit
#> K2         0.01188901   872.209                  TRUE
#> K3         0.01187201   846.389                  TRUE
plot_df <- data.frame(
  K = factor(summary_table$K),
  strict = summary_table$max_abs_prob_delta,
  warm = summary_table$warm_max_abs_prob_delta
)
plot_df <- reshape(
  plot_df, varying = c("strict", "warm"),
  v.names = "max_abs_prob_delta",
  timevar = "route", times = c("strict_joint", "warm_ovr"),
  direction = "long")
ggplot(plot_df, aes(K, max_abs_prob_delta, fill = route)) +
  geom_col(position = "dodge") +
  scale_y_log10() +
  labs(x = "K mode", y = "max absolute probability delta") +
  theme_minimal(base_size = 11)

Joint softmax probability error versus warm OVR baseline.

The strict joint route reaches maximum class-probability gaps of 7.658e-04 across K = 2 and K >= 3, while the warm OVR route is around 2.443e-01 on the same fixture. Both returned-object audits pass.
